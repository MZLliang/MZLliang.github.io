2.数据类型：Java常用数据类型有种：int double char String boolean
3.变量声明及使用：声明一个变量就是给变量赋初值。变量必须经过声明和赋值才能使用。

* 变量没有初始化

  • 关系运算符：关系运算符的值为布尔类型，只有两种结果：true和flash。
    < > <= >= != ==
    =和==的区别：
    =为赋值运算，而==则是比较运算符

    数据类型转换包括自动转换和强制转换，发生自动类型转换时必须符合一定的条件。

第三章总结：

Java中的if选择结构：包括以下几种形式：
基本if选择结构：可以处理简单的条件分支情况；
if-else选择结构：可以处理简单的条件分支情况；

第四章：

1.switch选择结构：用来判断条件为等值的选择结构
2.为什么使用switch选择结构：

  • switch表示开关，这个开关就是switch后边括号里面的值，小括号内放一个变量（表达式）或者字符型变量。
  • case表示情况，case后必须是一个整型或字符型的常量，通常是一个固定的字符数字，
  • default表示默认，即其它情况都不满足的条件下执行。
  • break表示停止，即跳出当前选择结构。

3.if和switch：
if用于执行某个变量处于连续的区域
switch只能处理等值条件的情况
4.选择结构总结：
基本if选择结构：
可以处理单一或者组合条件的情况。
5.if-else：
可以处理简单的条件分支情况。
6.多重if选择结构：
用于处理复杂的条件分支情况，多重if选择结构在解决需要判断的条件是连续的区间有很大的优势。
7.嵌套if选择结构：在if选择结构中包含一个或多个if选择结构，这种接哦古一般用于复杂的程序控制中（建议最多3层）

第四章总结：

当需要多重分支，并且条件为等值的条件下，用switch，当判断条件为连续的区间的情况下用if，在使用switch的时候一定要加break。
为了增加程序的健壮性，可以在程序中主动做出判断，并给用户提示，基本if选择结构，可以处理单一或者组合条件的情况。

第五章：

1.为什么需要while循环;
当满足条件时需要做什么事情就需要用到while循环。
2.什么时while：

3.while循环步骤：
(1)声明并初始化循环变量
(2)判断循环条件，当条件为真时，执行循环体，当条件为false时，直接退出循环
(3)执行循环操作后，再次判断循环条件
常见错误：
(1)没有初始化循环变量
(2)缺少{}
(3)没有结束循环的语句，造成死循环
4.什么程序调试：
找出系统中的问题所在，在程序需要的地方暂停，以便查看运行到这里时的变量值，希望单步运行程序，跟踪程序的运行流程，观察那条语句执行了，那条语句没有执行，满足暂停程序，观察变量和逐条执行语句等功能的工具和方法总称为程序调试。
5.程序调试的流程：
(1)分析错误，设置断点（双击主要部分设置断点）
(2)启动程序，单步执行 （单击运行旁边的debug虫子型按钮，执行）
(3)单步执行（f5为单步跳入：会进入本行的代码内部执行 f6为单步执行：仅执行本行代码，执行完后跳到下一行代码）
6.什么是do-while：
当一开始条件不满足的时候，while一次也不会执行，有时有这样的需要:无论如何都先执行一次，再判断循环条件，决定是否在继续执行，这就是do-while。

7.执行过程为;
(1)声明并初始化循环变量
(2)执行一遍循环体
(3)判断循环条件是否成立，成立则继续执行循环体，不成立则退出。
8.如何使用do-while：
(1)分析循环条件和循环操作
(2)套用do-while语句写出代码
(3)检查循环是否能退出。
9.while和do-while的区别：
(1)语法不同
(2)执行次序不同
(3)while可能一次也不执行，而do-while至少无条件执行一次

第五章总结：

循环结构有循环条件和循环操作构成，只要满足循环条件，循环操作就会反复执行。
使用循环解决问题的步骤：分析循环条件和循环操作，套用循环的语法写出代码，检查循环能否退出。
编写循环结构时需注意：循环变量的初值，循环操作中对循环变量值的改变和循环条件三者间的关系；确保循环次数正确，不要出现死循环。
while循环可能一次也不执行，但do-while一定会无条件执行一次。
程序调试是满足暂停程序，观察变量和逐步执行语句等功能的工具和方法的总称，其主要方法包括设置断点，单步执行和观察变量。

第六章：

1.for循环：
循环语句的作用是反复执行一行代码，直到满足一定的条件为止。
2.为什么需要for循环：
在解决有固定的循环次数的时候可以用for循环。
3.for循环结构：
(1)初始部分
(2)循环条件
(3)循环体：反复执行的代码
(4)迭代部分：下一次循环开始前要执行的部分，即结束循环的语句。
4.for循环的语法格式：

5.for循环的执行顺序：
执行初始部分—->进行条件判断——>根据判断的结果执行循环体——>执行迭代部分，改变循环变量的值—–>依次重复步骤2-3，直到退出for循环
6.使用for要注意：
在for循环中有3个表达式，在语法上3个表达式都可以省略，但分号不能省略，如果省略了表达式，要注意保证循环能够正常运行。
注意退出的循环条件，不要造成死循环。
7.转语句：
跳转语句有3中：break；continue；return；
(1)break：终止某个循环，程序跳转到循环体外的下一条语句，只要用switch中。
(2)continue：跳出本次循环，开始下次循环。
(3)return：从当前的方法中退出,返回到该调用的方法的语句处,继续执行

第六章总结;

while:先进行条件判断，在执行循环。如果条件不成立，退出循环。
do-while:先执行循环体，在进行条件判断，循环体至少执行一次。
for:先执行初始化部分，在进行条件判断，然后在执行循环体，最后进行迭代部分的计算，如果条件不成立，跳出循环。

第十一章：

1.对象：
对象是在其类模型的基础上构造出的，是类的具体实例，用来描述客观事物的一个实体，由一组属性和方法构成。
2.类：
是一个模板，描述了同一类对象都具有的数据和行为。
3.封装：
就是把一个事物包装起来，并尽可能隐藏内部细节。
4.类和对象的关系:
类和对象的关系是具体和抽象的关系。类是对象的概念模型，对象是一个类的实例。
5.Java的类模板：

6.类的命名遵循规则：

  • 不能使用Java中的关键字
  • 不能包含任何嵌入的空格或点号，以及下划线-&字符外的特殊字符
  • 不能以数字开头

main()方法是程序的入口，可以出现在任何要给类中，但要保证一个Java类中只有一个main方法，目的是实现不同的功能。
在Java中，要引用对象那个的属性和方法，需要使用“.”操作符，其中，对象名在圆点的左边，属性或方法的名称在圆点的右边。
9.面向对象：
是基于面向过程而言的，就是说面向对象是将功能等通过对象来实现，将功能封装进对象之中，让对象去实现具体的细节；这种思想是将数据作为第一位，而方法或者说是算法作为其次，这是对数据一种优化，操作起来更加的方便，简化了过程。面向对象有三大特征：封装性、继承性、多态性，其中封装性指的是隐藏了对象的属性和实现细节，仅对外提供公共的访问方式，这样就隔离了具体的变化，便于使用，提高了复用性和安全性。
10.面向对象的优点：
(1)与人类思维习惯一致，面向对象的思维的方式是从人类的角度出发，把人类解决问题的思维过程转变为程序能理解的过程。
(2)信息隐藏：提高系统的可维护性和安全性，封装实现了模块化和信息隐藏。
(3)提高系统的可重用性

第十一章总结：

对象是用来描述客观事物的一个实体，有一组方法和属性构成。类定义了对象将会拥有的特征（属性）和行为（方法）。
类和对象的关系：是抽象和具体的关系，类是对象的类型，对象是类的实例。
对象的属性和方法被共同封装在类中，相辅相成，不可分割。
面向对象程序设计的优点如下：
* 与人类思维一致。
* 隐藏信息，提高系统的可维护性和安全性
* 提高了系统的可重要性
